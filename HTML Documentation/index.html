Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create webpages. HTML structures content using tags and elements.

This tutorial platform teaches HTML from beginner basics to advanced concepts with examples and exercises.

Glossary

Tag
An HTML keyword inside angle brackets.
Element
A complete HTML structure with opening and closing tags.
Attribute
Extra information added to HTML tags.
