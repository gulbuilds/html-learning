What is HTML?

HTML is used to structure webpages. 


<h1>Hello World</h1>

Output: Displays a heading.

HTML Document Structure


<!DOCTYPE html>
<html>
<head>
    <title>Page</title>
</head>
<body>
    Content
</body>
</html>

Headings


<h1>Main Heading</h1>
<h2>Sub Heading</h2>

Paragraphs


<p>This is a paragraph.</p>

Links


<a href="https://example.com">Visit</a>

Images


<img src="image.jpg" alt="Sample">

Lists


<ul>
    <li>Apple</li>
    <li>Banana</li>
</ul>

Tables


<table border="1">
    <tr>
        <th>Name</th>
    </tr>

    <tr>
        <td>Ali</td>
    </tr>
</table>

Comments


<!-- This is comment -->

Attributes


<a href="page.html">Open</a>

Common Mistakes

  • Forgetting closing tags
  • Wrong nesting of tags
  • Missing alt attributes
