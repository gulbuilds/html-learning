What is an Anchor Tag?

Rule: The 'a' tag is used to create hyperlinks. 'a' stands for "anchor". The 'href' attribute contains the URL.

Examples:

1.External Link

Visit Google: Click here for Google

Note: target="_blank" opens link in new tab.

2.Link to My Other Page

3.Email Link
